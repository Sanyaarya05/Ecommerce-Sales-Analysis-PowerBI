# Ecommerce-Sales-Analysis-PowerBI

## Project Overview
This project analyzes e-commerce sales data to uncover insights related to revenue, profit, customer behavior, and payment modes.

## Key Features
- KPI tracking: Sales, Profit, Quantity, AOV
- Monthly profit trend analysis
- State-wise and category-wise performance
- Interactive slicers for dynamic analysis

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Key Insights
- COD contributes the highest share of profit
- Clothing category generates maximum profit
- Certain months show negative profit indicating seasonal impact
